# **PANDA is built on a modified gantry: [PROVerXL 4030 V2](https://www.sainsmart.com/products/proverxl-4030-v2)**

1. Assemble gantry EXCEPT for the deck
2. Lasercut or mill your material of choice (we used polycarbonate, 9 mm thick) using the following template: [documentation/3D-prints/PandaDeck.step](https://github.com/BU-KABlab/PANDA-BEAR/blob/2f304224014412291cdf1b685c452a97c4ee19d8/documentation/3D-prints/PandaDeck.step)
3. Install modified deck
4. 3D print deck accessories (as needed): [documentation/3D-prints/DeckAccessories](https://github.com/BU-KABlab/PANDA-BEAR/tree/2f304224014412291cdf1b685c452a97c4ee19d8/documentation/3D-prints/DeckAccessories)
5. 3D print paw components (as needed): [documentation/3D-prints/PAW](https://github.com/BU-KABlab/PANDA-BEAR/tree/2f304224014412291cdf1b685c452a97c4ee19d8/documentation/3D-prints/PAW)



See the bill of materials for a comprehensive list of all the parts used for the PANDA-film system:
[PANDA-film_BillOfMaterials.pdf](https://github.com/user-attachments/files/23193265/PANDA-film_BillOfMaterials.pdf)
